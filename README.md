# Loja de Pontos Back-End

Back-End para gerenciar um site de loja de produtos e controle de pontos

## Stack utilizada

**Front-end:** Vue, Vuetify, Vuex, Axios.

**Back-end:** NodeJs, Sequelize, PostgreSQL, Express, Body-parser, Cors, JWT.
